## Telas do projeto

### 1. Tela de Login
- Campos: Usuário e Senha
- Usuário válido: `admin`
- Senha válida: `123456`
- Se credenciais corretas → avança para tela de Menu
- Se incorretas → exibe mensagem "Usuário inválido"

### 2. Tela de Menu
- Botão "Cálculo de IMC" → abre tela de IMC  
- Botão "Equipe" → abre tela de Equipe  
- Botão "Voltar para Login" → retorna para a tela de Login  

### 3. Tela de IMC
- Campos: Nome, Peso (kg) e Altura (m)
- Botão "Calcular" → calcula IMC e mostra mensagem com o nome + valor do IMC
- Botão "Voltar" → retorna para o Menu

### 4. Tela de Equipe
- Exibe os nomes dos integrantes
- Botão "Voltar" → retorna para o Menu

## Como executar o projeto
1. Abrir o projeto no Android Studio (Giraffe ou superior).
2. Sincronizar as dependências (`Sync Now`).
3. Rodar o app em um emulador ou dispositivo físico com Android 7.0 (API 24) ou superior.
